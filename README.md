# com103 midterm exam
How the code works?

This is a simple Python program that helps a group track their project tasks.

## Description
The program shows a list of tasks with their estimated hours. It lets the user assign tasks to group members and set
the status as done or pending. Each task gives points based on its status.
After all tasks are entered, the program calculates the total points and the overall progress of the group.
It also shows the project status like "PROJECT READY!", "ON TRACK.", or "NEEDS MORE WORK!".

At the end, it displays a task board with all the details such as tasks, assigned members, status, points, and progress.


# How to Run
1. Make sure Python is installed  
2. Download the file midterm_solution.py  
3. Run in terminal:
   python midterm_solution.py
